深度学习入门:DL04-激活函数

发表于2024-05-15|更新于2024-06-08|深度学习入门
|阅读量:

在深度学习的世界里，激活函数是神经网络的非线性引擎，它们赋予了网络处理复杂数据和模式识别的能力。

今天，我们将深入探讨激活函数的基础知识，包括它们的由来、作用以及在神经网络中的应用。

一、什么是激活函数？

激活函数的概念受到生物神经系统的启发。在生物学中，神经元通过一个阈值机制来决定是否应该“发射”或传递信号。

类似地，人工神经网络中的激活函数就像是一个个小小的“开关”，它们决定了一个信号是否足够“响亮”，以至于能够被网络中的下一个神经元接收到。

二、为什么需要激活函数？

想象一下，如果没有这些“开关”，无论输入信号多么微弱，都会被直接传递下去，这样网络就无法区分哪些信号是重要的，哪些不是。激活函数的引入，让网络有了区分和处理不同信号的能力。

线性与非线性的较量：早期的神经网络模型大多基于线性假设，这限制了它们处理非线性问题的能力。引入激活函数后，神经网络能够模拟更加复杂的非线性关系，处理各种复杂的模式识别任务。

三、常见的激活函数

本文将介绍四种常用的激活函数：Sigmoid、Tanh、ReLU、Leaky ReLU和Softmax，从数学表达式、函数图像、优点、缺点和应用场景五个维度进行探讨，为您提供关于激活函数的全面理解。

3.1 Sigmoid函数

★数学表达式：
$$f(x) = \frac{1}{1+e^{-x}}$$
其中，$e$是自然常数，$x$是输入。

★函数图像：

  • 特性：
    • 非线性：Sigmoid 函数是S形的，它提供了一种将输入的任意实数值压缩映射到(0, 1)区间内的方式，从而引入非线性。
    • 平滑性：Sigmoid函数的导数具有良好的平滑性质，这使得它在反向传播算法中易于使用。
    • 渐近特性：当输入值趋向于正无穷或负无穷时，Sigmoid函数的输出分别趋向于1或0，这意味着它具有“开关”功能。
  • 优点：
    • 简化二分类：Sigmoid的输出可以被解释为概率，这使得它在二分类问题中非常自然，其中类别标签经常被视为0和1。
    • 避免饱和：与某些其他激活函数相比，Sigmoid在输入值绝对值较大时仍然有非零的梯度，尽管这个梯度可能非常小。
  • 缺点：
    • 梯度消失：当输入值绝对很大时，Sigmoid函数的梯度会接近于0，这会导致在反向传播过程中梯度值变得非常小，从而减慢学习过程，这种现象称为梯度消失。
    • 计算资源：相比于ReLU等激活函数，Sigmoid函数涉及指数运算，计算成本更高。
    • 非零中心化：Sigmoid函数的输出不是以0为中心的，这意味着在神经网络的下一层，神经元的权重更新将不得不处理非中心化的输入分布，这可能导致学习效率降低。
  • 应用场景：
    尽管存在梯度消失问题，Sigmoid仍然在特定场景下被使用，尤其是在二分类问题中，如逻辑回归模型。此外，它也常作为其他复杂激活函数的一个组成部分。

在设计神经网络时，选择激活函数是一个重要的决策，需要根据问题的特性和网络结构的需求来做出选择。尽管Sigmoid在某些情况下可能不是最优选择，但它在深度学习和机器学习的历史中仍然占有一席之地。

3.2 Tanh（双曲正切）函数

★数学表达式：
$$f(x) = \frac{e^x-e^{-x}}{e^x+e^{-x}}$$
其中，$e$是自然常数，$x$是输入。

★函数图像：

  • 特性：
    • 零中心化：Tanh函数的输出范围在-1到1之间，这使得它具有零中心化的特性，有助于数据的归一化处理。
    • 表达能力强：由于其S形曲线，Tanh能够表达输入值的正负变化，这在某些情况下可以提供比ReLU更强的表达能力。
  • 优点：
    • 数据归一化：Tanh的输出是零中心化的，这有助于网络在训练过程中保持数据在一个稳定的范围内，从而可能加快收敛速度。
    • 表达负值：与Sigmoid相比，Tanh能够表达负值，这在某些类型的任务中可能是一个优势。
  • 缺点：
    • 梯度消失：与Sigmoid类似，当输入值绝对较大时，Tanh的梯度会变得非常小，这会导致梯度消失问题，使得网络难以学习。
    • 计算复杂度：相比于ReLU，Tanh涉及指数运算，因此在计算上更加昂贵。
    • 不适用于输出层：在二分类问题的输出层，通常不使用Tanh，因为它的输出范围是-1到1，而不是0到1，这可能会使得概率解释变得模糊。
  • 应用场景：
    尽管Tanh存在梯度消失的问题，但它在某些特定的场景下仍然有其用途。例如，在某些特定的神经网络结构中，或者在数据需要被归一化到-1到1区间的情况下。

在实际应用中，由于ReLU及其变种的高效性和相对较少的梯度消失问题，它们比Tanh更受欢迎。然而，Tanh作为一个经典的激活函数，对于理解神经网络的工作原理和历史发展仍然具有重要意义。

在选择激活函数时，需要考虑网络的特定需求、问题的类型以及希望避免的问题（如梯度消失）。实验和交叉验证通常是确定最佳激活函数的有效方法。

3.3 ReLU (Rectified Linear Unit)函数

★数学表达式：
$$f(x)=max(0,x)=\begin{cases} x, & if(x＞0) \ 0,& if(x≤0) \ \end{cases}$$

其中，$x$是输入。

★函数图像：

  • 特性：
    • 线性：对于所有正输入值，ReLU是一个线性函数，这使得它在正区间内易于优化。
    • 计算简单：ReLU的计算非常直观和简单，因为它只涉及一个阈值操作。
  • 优点：
    • 计算效率高：ReLU相比于Sigmoid或Tanh，不涉及复杂的指数运算，因此在计算上更加高效。
    • 缓解梯度消失问题：对于正输入，ReLU的梯度是恒定的，这有助于缓解梯度消失问题，加速收敛。
    • 稀疏激活：由于ReLU在负输入值处的输出为0，它能够在网络中产生稀疏激活，这有助于特征选择和减少计算量。
  • 缺点：
    • 神经元死亡：对于负输入值，ReLU的梯度为0，这可能导致在训练过程中某些神经元永远不会激活，即所谓的神经元死亡问题。
    • 不连续：在x=0处ReLU函数不连续，这可能会在优化过程中造成一些不稳定性。
    • 未中心化：与Sigmoid或Tanh不同，ReLU不是一个零中心化的激活函数，这意味着输出数据的分布可能不会围绕0中心化。
  • 应用场景：
    ReLU由于其计算简单和训练效率高，在深度学习中被广泛用于隐藏层。它特别适合用于大型网络和大数据集，因为这些情况下梯度消失问题尤为严重。

为了解决ReLU的一些缺点，如神经元死亡，已经提出了一些改进版本：

  • Leaky ReLU：允许负输入有一个非零的梯度。
  • Parametric ReLU (PReLU)：Leaky ReLU的参数化版本，其中泄露率是可学习的。
  • Exponential Linear Unit (ELU)：对于负值，提供一个非零的平滑梯度。

在实际应用中，ReLU及其变体是现代深度神经网络中非常流行的激活函数之一，尤其是在处理图像数据的卷积神经网络（CNN）中。然而，选择哪种激活函数仍然需要根据具体问题和数据特性来决定。

3.4 Leaky ReLU函数

★数学表达式：

$$f(x)=\begin{cases} x, & if(x＞0) \ \alpha x,& if(x≤0) \ \end{cases}$$

其中，$\alpha$是一个小于1的常数，通常取一个很小的值（比如0.01）；$x$是输入。

★函数图像：

  • 特性：
    • 非零梯度：Leaky ReLU在负输入值处仍然有一个非零的梯度，这有助于缓解标准ReLU中的神经元死亡问题。
  • 优点：
    • 缓解神经元死亡：由于在负输入值处有一个非零的梯度，Leaky ReLU可以保证即使输入为负，神经元也会被激活，只是激活程度较低。
    • 计算简单：Leaky ReLU的计算与ReLU相似，非常高效。
    • 稀疏激活：与ReLU一样，Leaky ReLU也支持稀疏激活，有助于特征选择。
  • 缺点：
    • 超参数：Leaky ReLU引入了一个新的超参数$\alpha$，这需要在模型训练前进行设置，增加了调参的复杂性。
    • 实现复杂性：相比于标准的ReLU，Leaky ReLU在实现上稍微复杂一些，因为它需要对正负输入值进行不同的处理。
  • 应用场景：
    Leaky ReLU在很多深度学习模型中被用作ReLU的一个改进版本，特别是在那些ReLU导致太多神经元死亡的场景中。它适合用于隐藏层，以引入非线性并保持网络的激活。

在实际应用中，Leaky ReLU由于其能够缓解神经元死亡问题，且计算效率高，被广泛用于深度神经网络的训练中。然而，选择哪种激活函数仍然需要根据具体问题和数据特性来决定，并且需要通过实验来验证其效果。

3.5 Softmax函数

★数学表达式：
$$f(x_i)=\frac{e^{x_i}}{\sum_{k=1}^{N} e^{x_k}}$$
其中，$e$是自然常数，$x_i$是输入。
★函数图像：暂无

  • 特性：
    • 概率分布：Softmax函数的输出可以被解释为概率分布，这使得它在处理多分类问题时非常有用。
    • 非线性：Softmax是一个非线性激活函数，可以增加神经网络的表达能力。
  • 优点：
    • 输出互斥：在多分类问题中，Softmax输出的各个类别概率互斥，即它们的和为1，这使得它适合用于表示类别概率。
    • 易于优化：Softmax损失（交叉熵损失）在多分类问题中是凸优化问题，有助于避免陷入局部最小值。
  • 缺点：
    • 数值稳定性：由于涉及指数运算，当输入值较大时，可能会遇到数值溢出的问题，通常需要通过减去一个常数（如输入的最大值）来解决。
    • 计算资源：相比于某些其他激活函数，Softmax的计算成本更高，因为它需要计算整个输入向量的指数。
  • 应用场景：
    Softmax激活函数最常用于神经网络的最后一层，尤其是在多分类问题中。例如，在处理图像识别任务时，如果目标是识别图像中的多个类别，输出层通常会使用Softmax函数。

Softmax激活函数与交叉熵损失函数一起使用，形成了一个强大的组合，用于训练多分类模型。在实际应用中，Softmax函数不仅帮助模型输出概率分布，还提供了一种自然的方式来计算损失，引导模型学习区分不同类别的特征。

在选择激活函数时，Softmax通常是多分类问题的首选，尤其是当输出层的类别数比较多时。然而，对于二分类问题，通常使用Sigmoid激活函数，因为二分类可以视为多分类的一个特例。

结语

激活函数是深度学习模型中的非线性引擎，它们使网络能够学习和执行复杂的任务。理解不同激活函数的特性和适用场景对于构建高效的神经网络模型至关重要。随着深度学习技术的不断发展，激活函数的设计和应用将继续是研究的热点。
