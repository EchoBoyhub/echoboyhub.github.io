什么是函数？

一、函数定义

设$x$和$y$是两个变量，$D$是一个非空数集；如果按照某个法则$f$，对每个变量$x∈D$，变量$y$总有唯一确定的值与之对应，则称此对应法则$f$为定义在$D$上的函数，与$x$对应的值$y$称为$f$在$x$处的函数值，记作$f(x)$，即$y=f(x)$，其中：
· 变量$x$称为自变量；
· 变量$y$称为因变量；
· 数集$D$称为函数定义域；
· 数集$W={y|y=f(x), x∈D}$称为自变量；

二、函数性质

1. 有界性

设函数$y=f(x)$的定义域为$D$；在集合$D$上，若存在正数$M$，使得$|f(x)|≤M$对于任意$x∈D$都成立，则称函数在$D$上有界；如果这样的$M$不存在，就称函数$f(x)$在$D$上无界。

等价于：无论对于任何正数$M$，总存在$x_1∈X$，使得$|f(x_1)|>M$，那么函数$f(x)$在$X$上无界。此外，函数$f(x)$在$X$上有界的充分必要条件是它在$X$上既有上界也有下界。

2. 单调性

（1）单调递增：如果$x_1,x_2∈I, x_1< x_2$，都存在$f(x_1)$$<$$f(x_2)$；

（2）单调递减：如果$x_1,x_2∈I, x_1< x_2$，都存在$f(x_1)>f(x_2)$。

3. 奇偶性

（1）偶函数：对于定义域中的任意$x$，都存在$f(-x)$$<$$f(x)$；

（2）偶函数：对于定义域中的任意$x$，都存在$f(-x)<-f(x)$；

其中，对于两个在定义域中有定义的函数：① 两个偶函数之和、之积均为偶函数；② 两个奇函数之和为奇函数，之积为偶函数；③ 一个奇函数与一个偶函数之积为奇函数。

4. 周期性

若存在正实数$T$，对于任意$x$，恒有$f(x+T)=f(x)$，则称$y=f(x)$为周期函数；

满足上述关系的最小正数$T$称为函数$f(x)$的周期。

三、常用函数

1. 幂函数：$y = x^\alpha(\alpha 是常数)$

2. 指数函数：$y=\alpha^x(\alpha＞0, \alpha≠1 )$

3. 对数函数：$y = log_\alpha x (\alpha＞0, \alpha≠1 )$

对数具有以下运算性质：对任意的$x, y∈R^+, a ＞0,a ≠1,b∈R$;

（1）$log_a x+log_a y=log_a xy$

（2）$log_a x-log_a y=log_a \frac{x}{y}$

（3）$log_a x^b=blog_ax$

4. 三角函数：$y=sinx$（正弦函数），$y=cosx$（余弦函数），$y=tanx$（正切函数）

5. 反三角函数：$y = arcsinx$，$y = arccosx$，$y = arctanx$

6. 常数函数：$y=C$

7. 绝对值函数：$y = |x| = \begin{cases} x, & x≥0 \ -x,& x＜0 \ \end{cases}$

8. 符号函数：$y = sgnx= \begin{cases} 1, & x＞0 \ 0, & x=0 \ -1,& x＜0 \ \end{cases}$
