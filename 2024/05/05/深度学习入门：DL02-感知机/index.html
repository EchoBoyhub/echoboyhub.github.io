深度学习入门:DL02-感知机

发表于 2024-05-05 更新于 2024-06-07 分类于 深度学习入门
本文主要介绍了一个最简单的深度学习模型——感知机。

在AI宇宙中，感知机（Rerception）犹如一颗璀璨的恒星，照亮了机器学习和深度学习的发展之路。感知机作为人工神经网络的最小单元，不仅是人工智能发展史上一个重要的里程碑事件，其模型的原理和思想至今仍对现代人工智能具有重要的意义。

接下来，我们就来聊一聊感知机的前世今生🤭吧！

一、感知机的发展

首先，为了更好的理解感知机，我们先来看一下感知机在神经网络发展历程（如下图）中的地位。深度学习是机器学习中的一个分支，而神经网络又是促进深度学习蓬勃发展的关键技术。但神经网络的发展还是具有非常悠久历史的，且发展历程可谓一波三折。

从上述发展历程中，我们得知罗森·布拉特受到”M-P模型”的影响发明了感知机，最早的人工神经网络，它是一种形式最简单的前馈神经网络，主要用于二元线性分类器。

感知机，作为神经网络的起点，不仅开启了AI新纪元，也成为我们理解神经网络本质的重要工具，为后续开发更复杂的神经网络奠定了基础。

二、什么是感知机？

2.1 感知机的工作原理

感知机的工作原理主要是基于人脑单个神经元处理信息的过程。它是一个简单的线性分类器，其结构如下图，由输入层、计算单元和输出层三个模块组成，它的输入是实例的特征向量，输出是实例的类别（通常取+1或-1）。

知识点详解：

  1. 输入层：感知机的输入层接收来自外部的输入信号，这些信号可以是特征值（例如，在逻辑”与”门的例子中，输入层接收是两个二进制值）。
  2. 权重：每个输入信号都有一个与之关联的权重。权重是感知机学习过程中需要调整的参数，用于量化输入信号对于最终决策的重要性。
  3. 偏置：感知机还包含一个偏置项，通常是一个常数，用于调整分类决策的阈值。偏置项为感知机提供了必要的灵活性，使其能够更好的拟合输入信号。
  4. 线性组合：输入信号，通过和其关联的权重相乘并进行线性组合，再加上偏置项，最终得到一个决策变量 $x$ ：$$z=w_1x_1+w_2x_2+\cdots+w_nx_n+b$$ 其中，$x_1, x_2, \cdots, x_n$是输入信号，$w_1, w_2, \cdots, w_n$是输入信号的权重，$b$是一个偏置项。
  5. 激活函数：感知机使用的是一个阈值激活函数，通常是Heaviside函数或符号函数，来确定最终的决策输出。激活函数将决策变量映射到一个二进制输出 $y$ ，数学公式：$$ y = \begin{cases} 0, & if z<0 \ 1,& if ≥ 0 \ \end{cases}$$ 或者使用符号函数来表示：$$y =sign(z)= \begin{cases} -1, & if z<0 \ +1,& if ≥ 0 \ \end{cases}$$
  6. 输出层：感知机的输出层产生最终的分类决策。通常是一个二进制值，表示输入信号是属于正类还是负类。
  7. 损失函数：感知机需要通过训练数据进行模型训练来确定参数值，而在训练过程中，损失函数可以准确的评估模型实际值和预测值之间的误差。（对于感知机而言，常用的损失函数是误分类损失，即0-1损失函数，它在样本被错误分类时为1，正确分类时为0）。
  8. 训练算法：感知机通过一种简单的训练算法来确定模型的参数，即感知机学习规则或Delta规则，在每次训练中，样本被错误分类后，感知机将通过更新权重和偏置来减少误分类的概率。
  9. 迭代优化： 感知机通过训练算法来多次迭代调整模型参数，直到模型能够将所有样本都正确分类，或达到设定的迭代次数为至。

通过上述的讲解，大家可能觉得感知机结构太简单，不值得花时间去学习。但你知道吗？感知机的结构是许多复杂深度学习算法的基础，它的训练算法也启发了很多更复杂模型的训练过程。

2.2 感知机的数学模型

数学模型：
感知机的数学模型表达式为：
$$f(z)=f(\sum_{i=1}^{n} w_i*x_i+b)=f(W^TX)$$
$$f(x) =sign(z)= \begin{cases} -1, & if z<0 \ +1,& if z≥ 0 \ \end{cases}$$

其中，

  1. 输入向量/特征向量：模型接收特性向量 $X=[x_0,x_1,x_2,\cdots,x_n]$ 作为输入，其中，$n$是特征的维度；$x_0$是偏置项，控制决策边界的位置。
  2. 权重向量：模型还需要一个权重向量 $W=[w_0,w_1,w_2,\cdots,w_n]$，其中每个权重 $w_i$ 对应一个特征 $x_i$。
  3. 线性方程组：模型通过计算特征向量和权重向量的线性方程组得到决策变量 $z=W^TX$ 或 $z=\sum_{i=1}^{n} w_i*x_i+b$。
  4. 激活函数：感知机使用阈值激活函数来确定最终的输出结果。常用的阈值激活函数就是Heaviside阶跃函数 $f(x)=sign(x)$，它根据决策变量 $z$ 输出类别标签。
