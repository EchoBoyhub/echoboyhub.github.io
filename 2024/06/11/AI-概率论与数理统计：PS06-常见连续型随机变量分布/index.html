在之前的文章中，我们学习了一维离散型和连续型随机变量的概率模型。但在许多实际问题中，我们需要使用多个随机变量来描述随机线性，比如，一般天气预报包括：空气质量、天气实况、温度、降水、风力、风向、气压、紫外线等，天气的描述需要多个随机变量。

所以，研究多维随机变量是要揭示各变量之间的相互联系和相互影响，这是研究多个一维随机变量分布无法得到的。而多维随机变量的研究方法和二维随机变量的研究思想及方法相同，为了简便起见，我们重点学习一下二维随机变量。

一、二维随机变量的定义

【定义】设有随机试验 $E$，其样本空间为 $\Omega$。若对 $\Omega$ 中的每一个样本点 $\omega$ 都有一对有序实数 $(X(\omega), Y(\omega))$ 与其对应，则称 $(X,Y)$ 为二维随机变量或二维随机向量。称 $(X,Y)$ 的取值范围为它的值域，记作$\Omega (X,Y)$。

通俗来说，二维随机变量是指在二维空间中定义的随机变量，通常用向量形式表示，比如 $(X,Y)$。这里的 $X$ 和 $Y$ 可以是独立的，也可以是相互依赖的。

二、联合分布函数

对于二维随机变量，我们同样也是要描述 $X$ 和 $Y$ 的概率分布，但是与一维随机变量不同之处在于 $(X,Y)$ 的分布不仅要包含每个随机变量各自的分布信息，还要包含两者之间相互关系的信息。因此称 $X,Y$ 的分布为联合分布。

【定义】设 $(X,Y)$ 为二维随机变量，对任意的 $(X,Y) \in R^2$， 称$$F(x,y)=P(X \leq x,Y \leq y)$$为随机变量 $(X,Y)$ 的（联合）分布函数。

上述定义中，$P(X \leq x,Y \leq y)$ 中的逗号表示对事件 ${X \leq x}$ 和事件 ${Y \leq y}$ 取积事件，即 $P(X \leq x,Y \leq y)=P({X \leq x} \bigcap {Y \leq y})=P((X,Y) \in D_{xy})$。其中 $D_{xy}$ 区域如下图所示。

三、离散型和连续型

3.1 二维离散型随机变量

二维离散型随机变量是指在二维空间中取值的随机变量，且这些取值是离散的、可数的。

二维离散型随机变量的联合概率分布由联合概率质量函数描述：$$p(x,y)=P(X=x,Y=y)$$
其中， $p(x,y)$ 是随机变量 $X$ 取值 $x$ 和随机变量 $Y$ 取值 $y$ 的联合概率。

边缘概率分布描述了单个随机变量的分布，可以通过对另一个变量的所有可能值求和来获得：

  • $p_X(x)=P(X=x)=\sum_y p(x,y)$
  • $p_Y(y)=P(Y=y)=\sum_x p(x,y)$

二维离散型随机变量 $(X,Y)$ 的分布函数是联合概率分布的累积形式。$$F(x,y)=P(X \leq x,Y \leq y)=\sum_{x’ \leq x} \sum_{y’ \leq y} p(x’, y’)$$

3.2 二维连续型随机变量

二维连续型随机变量是指在二维空间中取值的随机变量，其取值范围是连续的，可以是任意实数。

二维连续型随机变量的联合概率密度函数 $f(x,y)$ 描述了随机变量 $X$ 和 $Y$ 同时取特定值的概率密度，$f(x,y)$ 定义为 $f(x,y) \geq 0$ 并且对于所有 $x,y$ 的积分满足 $\int^{\infty}_{-\infty} f(x,y)\mathrm{d}x \mathrm{d}y=1$。

其中， 随机变量 $X$ 和 $Y$ 同时落在 $(x,x+dx)$ 和 $(y,y+dy)$ 内的概率近似为 $f(x,y)\mathrm{d}x \mathrm{d}y$

边缘概率密度函数描述了单个随机变量的概率分布，可以通过对另一个变量的联合概率密度函数进行积分来获得：

  • $f_X(x)=\int^{\infty}_{-\infty} f(x,y)\mathrm{d}y$
  • $f_Y(y)=\int^{\infty}_{-\infty} f(x,y)\mathrm{d}x$

二维连续型随机变量的联合概率分布函数 $F(x,y)$ ：$$F(x,y)=P(X \leq x,Y \leq y)=\int^x_{-\infty} \int^y_{-\infty} f(u, v) \mathrm{d}u \mathrm{d}v $$

总结

二维连续型随机变量是概率论和统计学中的非常重要的概念，它们帮助我们深入理解并分析两个连续变量之间的关系。
