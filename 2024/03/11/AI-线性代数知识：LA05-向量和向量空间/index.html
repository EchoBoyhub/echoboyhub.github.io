AI数学知识：LA05-向量和向量空间

发表于 2024-03-11 更新于 2024-07-06 分类于 AI数学知识 ， 线性代数
本文介绍了线性代数中向量和向量空间的概念。

引言

大家好，今天我们来继续探索向量的世界，一个充满数学魅力的领域。向量不仅仅是数学课上的抽象概念，它们在我们的日常生活中无处不在，从物理学的力和速度，到计算机图形学中图像的变换，都离不开向量的应用。

现在，让我们一起来了解向量的内积、长度、正交性以及向量空间的基。

1. 向量的内积

向量的内积，也称为点积或数量积，是两个向量之间的一种运算，它返回一个标量（即一个数值，而不是向量）。

内积在数学、物理学和工程学中有着广泛的应用。

💡内积的定义：

两个向量的内积定义为：

  1. 两个向量的对应分量乘积的总和。
  2. 一个向量的长度与其在另一个向量方向上的投影的乘积。

🧭内积的计算：

对于两个在$n$维空间的向量：$\textbf{x}=\begin{pmatrix}x_1 \ x_2 \ \cdots \ x_n \end{pmatrix}$, $\textbf{y}=\begin{pmatrix}y_1 \ y_2 \ \cdots \ y_n \end{pmatrix}$.

若有$[\textbf{x},\textbf{y}] = x_1y_1+x_2y_2+\cdots+x_ny_n$, 此时我们就把$[\textbf{x},\textbf{y}]$叫做向量的内积。
即，向量表示形式：$[\textbf{x},\textbf{y}]=(x_1,x_2,\cdots,x_n)\begin{pmatrix}y_1 \ y_2 \ \cdots \ y_n \end{pmatrix}=\textbf{x}^T\textbf{y}$.

二维空间：$\textbf{u} \cdot \textbf{v}$ = $|u||v|\cos(\theta)$, 其中 $|u|$ 和 $|v|$ 分别是向量 $\textbf{u}$ 和 $\textbf{v}$ 的长度，$\theta$是两个向量之间的夹角。

📌内积的性质：

  1. 对称性：$[\textbf{x},\textbf{y}]=[\textbf{x},\textbf{y}]$.
  2. 线性性质：$[\lambda \textbf{x},\textbf{y}]=\lambda[\textbf{x},\textbf{y}]$, $[\textbf{x}+\textbf{y},z]=[\textbf{x},\textbf{z}]+[\textbf{y},\textbf{z}]$.

内积是一个在许多领域中都有重要应用的基本概念，它提供了一种量化两个向量之间关系的方法。

注：本文用加粗的字母表示向量，即$\textbf{x}$：表示向量$x$ 或 $\overrightarrow{x}$.

2. 向量的长度

向量的长度，也称为向量的模或大小，是一个表示向量从起点到终点的直线距离的标量。在不同的上下文中，向量长度的计算方式可能有所不同，但最常见的是在欧几里得空间中，起计算方式

🧭向量长度的计算：

在 $n$ 维空间中，向量$\textbf{x}$的长度：$||\textbf{x}||$=$\sqrt{[\textbf{x},\textbf{x}]}$=$\sqrt{x_1^2+x_2^2+\cdots+x_n^2}≥0$.
其中，$x_1,x_2,\cdots,x_n$是向量$\textbf{x}$在各个维度上得分量。

当$||x||=1$时，向量$x$称为单位向量。

📌向量长度的性质：

  1. 齐次性：$||\lambda \textbf{x}||=|\lambda|\cdot||\textbf{x}||$.
  2. 三角不等式：$||\textbf{x}+\textbf{y}||≤||\textbf{x}||+||\textbf{y}||$.

    向量的长度可以直观地理解为从原点到向量顶点的直线距离。在二维平面上，你可以想象成一个由原点、向量顶点和向量本身构成的直角三角形的斜边长度。

🧩向量的标准化：

向量的标准化是将向量的长度变为1的过程，得到的向量称为单位向量。单位向量的方向与原向量相同，但长度为1。

一个向量 $\textbf{v}$ 的标准化形式 $\hat{\textbf{v}}$ 可以通过公式得到： $\hat{\textbf{v}}=\frac{\textbf{v}}{||\textbf{v}||}$

如果 $\textbf{v}$ 不是零向量，那么 $\hat{\textbf{v}}$ 就是 $\textbf{v}$ 的单位向量。

向量长度的概念是向量代数的基础之一，它在数学的许多分支以及物理学和工程学中都有重要应用。

3. 向量的正交

正交性是线性代数中的一个重要概念，它在数学、物理学、工程学以及计算机科学等多个领域中都有广泛的应用。

🧭向量正交的概念：

在向量空间中，两个向量如果它们的点积为零，则称这两个向量是正交的。换句话说，如果有两个向量 $\textbf{u}$ 和 $\textbf{v}$ , 它们的点积$\textbf{u} \cdot \textbf{v}=0$, 则 $\textbf{u}$ 和 $\textbf{v}$ 是正交的。

几何意义：在二维或三维空间中，如果两个向量 $\textbf{u}$ 和 $\textbf{v}$ 正交，那么它们之间的夹角是$90^o$（直角）。这意味着一个向量的方向完全垂直于另一个向量的方向。

📌向量正交的性质：

  • 若$\textbf{a}_1,\textbf{a}_2,\cdots,\textbf{a}_r$是两两正交的非零向量，则$\textbf{a}_1,\textbf{a}_2,\cdots,\textbf{a}_r$线性无关。
  • 正交基：如果一组基向量两两正交，那么这组基向量称为正交基。如果这组基向量的模长都是1，那么它们构成的是标准正交基。
  • 正交投影：正交性用于计算一个向量在另一个向量上的正交投影。正交投影的长度表示两个向量之间在垂直方向上的“距离”。
  • 最小二乘法：在数据拟合中，正交性用于最小化误差的平方和，寻找最佳拟合线或面。
  • 傅里叶分析：在信号处理中，正交性用于将信号分解为不同频率的正交分量。

正交性提供了一种强大的工具，用于解决涉及垂直性和独立性的问题，是理解和应用许多数学和工程概念的基础。

4. 向量空间的基

在向量空间中，”基“（Basis）是一个非常重要的概念，它为空间中的每个向量提供了一种独特的表示方式。

💡”基”的定义：

向量空间的基是该空间中一组线性无关的向量，它们满足以下条件：

  1. 线性无关：基中的向量不能通过其他向量的线性组合来表示。
  2. 生成空间：基向量的任意线性组合能够生成整个向量空间。

🧭”基”的性质：

  • 唯一性：向量空间中的每个向量都可以唯一地表示为基向量的线性组合。
  • 维数：基向量的数量决定了向量空间的维数。一个向量空间的维数是该空间中所有基的共同向量数。
  • 扩展性：如果一个向量集合可以构成基，那么任何包含这个集合的向量集合，如果还能构成基，那么新基的向量数量不会减少。

示例：1. 在二维空间$\mathbb{R}^2$中，基可以是$e_1 =\begin{pmatrix}1 \ 0 \end{pmatrix}$、$e_2 =\begin{pmatrix}0 \ 1 \end{pmatrix}$,即水平和垂直的单位向量。
2. 在三维空间$\mathbb{R}^3$中，基可以是$e_1 =\begin{pmatrix}1 \ 0 \ 0 \end{pmatrix}$、$e_2 =\begin{pmatrix}0 \ 1 \ 0 \end{pmatrix}$、$e_3 =\begin{pmatrix}0 \ 0 \ 1 \end{pmatrix}$, 即三个坐标轴的单位向量。

🧩正交基和规范正交基：

  • 正交基：如果基中的每个向量都两两正交（即它们的点积为零），则这个基称为正交基。

  • 规范正交基：不仅正交，而且每个向量的模长都是1，这样的基称为标准正交基。

    设$n$维向量$\textbf{e}_1,\textbf{e}_2,\cdots,\textbf{e}_r$是向量空间$V \in \mathbb{R}^n$中的向量。满足以下条件：

    • $\textbf{e}_1,\textbf{e}_2,\cdots,\textbf{e}_r$是向量空间$V$中的一个基；
    • $\textbf{e}_1,\textbf{e}_2,\cdots,\textbf{e}_r$两两正交；
    • $\textbf{e}_1,\textbf{e}_2,\cdots,\textbf{e}_r$都是单位向量；

    则称$\textbf{e}_1,\textbf{e}_2,\cdots,\textbf{e}_r$是$V$的一个规范正交基。

    例，$e_1 =\begin{pmatrix}1 \ 0 \ 0 \end{pmatrix}$、$e_2 =\begin{pmatrix}0 \ 1 \ 0 \end{pmatrix}$、$e_3 =\begin{pmatrix}0 \ 0 \ 1 \end{pmatrix}$ 是$\mathbb{R}^3$的一个规范正交基。

“基“的概念是理解和操作向量空间的基础，它在数学、物理学、工程学和计算机科学等领域中都有着广泛的应用。通过基，我们能够以一种标准化和系统化的方式来处理向量空间中的各种问题。

结语

通过这篇文章，我们了解了向量的内积、长度、正交性和向量空间的基这些基本概念。这些概念不仅在数学领域内非常重要，它们也是我们理解和探索世界的重要工具。希望这篇文章能够帮助你更好地理解向量和向量空间，并激发你对数学和人工智能的兴趣。
